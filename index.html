واجب Snort - مراقبة الشبكة وإعطاء Alert

جامعة تكريت - كلية علوم الحاسوب والرياضيات - قسم الأمن السيبراني

الطالب: عبدالله احمد مطلك

1. التأكد من تثبيت Snort

افتح Terminal واكتب:

snort -V

إذا ظهر الإصدار، Snort مثبت ✔️

2. معرفة كرت الشبكة

لتحديد واجهة الشبكة النشطة:

ip a

3. إنشاء Rule لإظهار Alert

افتح ملف القواعد:

sudo nano /etc/snort/rules/local.rules

أضف السطر التالي لاكتشاف Ping (ICMP):

alert icmp any any -> any any (msg:"ICMP Ping Detected"; sid:1000001; rev:1;)

4. تشغيل Snort

نفذ الأمر التالي لمراقبة الشبكة وعرض التنبيهات على التيرمنال:

sudo snort -i eth0 -c /etc/snort/snort.conf -A console

5. اختبار Alert

من جهاز آخر أو نفس الجهاز:

ping 8.8.8.8

ستظهر رسالة Alert في التيرمنال مثل:

[**] [1:1000001:1] ICMP Ping Detected [**]

الخاتمة

تمت مراقبة الشبكة باستخدام Snort وإظهار تنبيه عند حدوث ICMP Ping. هذا يوضح قدرة Snort على العمل كـ IDS لمراقبة الشبكات واكتشاف الأنشطة المشبوهة.
